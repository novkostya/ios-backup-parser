# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal
attacks are not tolerated in issues, pull requests, or any other project space.

To report unacceptable behavior, contact the maintainer via GitHub
([@novkostya](https://github.com/novkostya)). Reports are handled confidentially.
